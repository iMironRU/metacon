# МетаКон

Браузерный тренажёр на **покрытие механизмов** платформы 1С:Предприятие 8.3 для
подготовки к экзамену «1С:Специалист». Тренирует навык **чтения и декомпозиции**
задачи (что справочник, что документ, что регистр…), а не зубрёжку решений.

- Концепция и обоснование: [`docs/CONCEPT.md`](docs/CONCEPT.md)
- Бриф для разработки: [`docs/HANDOFF.md`](docs/HANDOFF.md)
- Контракт карточки: [`schema/mechanism-card.schema.json`](schema/mechanism-card.schema.json)
- Seed-эталон: [`cards/operativny/partii_fifo_lifo.json`](cards/operativny/partii_fifo_lifo.json)

## Инварианты

1. **Карточки — данные, движок — код, разделены.** Карточки = JSON по схеме; чекеры =
   чистые функции `(карточка, ответ) → вердикт`.
2. **MVP безсерверный.** Уровни 1–2 грейдятся в браузере детерминированно; статика → GitHub Pages.
3. **Один `model` → проекции уровней.** Размечаем модель один раз, уровни — её проекции.

Стек: Vite + React + TypeScript.
